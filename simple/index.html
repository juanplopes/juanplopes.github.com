Instructions

There are many ways to start using Simple.Net in your applications. We're gonna cover two of them.

The easy way

  1. Download (or clone) the repository at github.
  2. Run build.cmd file
  3. Type 'package' and press ENTER
  4. Find the exe file inside the 'pkg' folder
